Teach Piano Online - Practice Sheet Generator and Random Measure Generator

The Practice Sheet Generator and Random Measure Generator are tools for teachers to create practice resources for their students.

Dependencies

  • VueJS: JavaScript framework and form handling
  • jsPDF: for the PDF generation
  • html2canvas: this package creates an image from a specific <div> that’s then put into the PDF
  • Moment.Js: used for the dates used
